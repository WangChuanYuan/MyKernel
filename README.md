# ls
Usage：ls [-adilR] [name...]

Options：  
-a: 显示隐藏文件(文件名以.开头的文件)  
-d: 显示目录本身信息，不列出目录条目  
-i：显示文件的inode number  
-l：显示文件的详细信息  
-R：递归显示目录子条目信息  

# wc
Usage：wc [filename...]  

Output：  
Entry format：line-count word-count byte-count filename  
include total when the number of arguments is greater than one
